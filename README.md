# Rust Tauri Text Editor

A lightweight text editor built with **Rust** and **Tauri**, featuring file creation, opening, saving, and folder browsing.

---

## Features

- **Create new files** quickly
- **Open existing files** for editing
- **Save files** with ease
- **Browse folder contents** directly within the app
- Lightweight and fast thanks to Rust and Tauri

---

## Tech Stack

- **Rust** – for the backend and core logic  
- **Tauri** – to build a cross-platform desktop application  
- **VS Code** – development environment
